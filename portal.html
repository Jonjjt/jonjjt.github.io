My Work on the Project

As this was a 2 person project, my partner and I worked together on most aspects of this project. These include ideation, concept development, arduino code and implementation with the electronics, and chassis design and construction. Nearer to the end of the project I worked on graphics and presentation materials while my partner continued work on the arduino code and electronics.
